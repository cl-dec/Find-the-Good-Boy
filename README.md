Find the Good Boy: The Python Text Game

In Find the Good Boy, the user must input a value from a list given to them until they guess the answer correctly. The correct answer is randomly generated from the elements in the list and is different every time the game is run. 

The object of the game is to find the user's dog who is trapped somewhere in the house. 
Inspired by Loki, the cat trapped in the drawer.
